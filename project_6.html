HOMEHOSPITAL MANAGEMENT SYSTEM




This project is basically developed for people to book an appointment with the doctor, then the doctor will approve or disapprove appointment booked by the user. If approved, then the doctor will leave a message to the user on how to contact him/her, otherwise a default message will be send to the user telling the user that the appointment is disapproved.
